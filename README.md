# iam-project
Built a browser-based Identity and Access Management console to simulate enterprise identity security workflows.  The application supports user provisioning, role-based access control, permission management, and audit logging for identity-related events.

## Features
- User management
- Role-Based Access Control (RBAC)
- Permission assignment
- Account status tracking
- Audit logging
- Search and filtering

## Security Concepts Demonstrated
- Identity lifecycle management
- Access control
- Role management
- Audit logging
- Authentication event visibility

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Preview
This project simulates an enterprise IAM console used to manage users, roles, and access events.
